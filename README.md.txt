# ElectroMart E-Commerce Project

This project contains a simple electronics e-commerce website.

## Folders:
- `frontend/` — HTML, CSS, and JS for the UI
- `backend/` — Node.js + MongoDB API for handling products

## To run backend:
1. Install dependencies: `npm install`
2. Add `.env` with your MongoDB URI
3. Run server: `nodemon app.js`
